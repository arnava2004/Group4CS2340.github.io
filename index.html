User Interface

Login

A streamlined login interface where existing users can enter their username and password. It includes a quit link and a prominent "Create Account" button for new users.

Personal Information

This crucial page collects essential user data to personalize the app experience. It includes name Age, Weight, Gender. The save information button submits all this info to the user database and updates the User's paramaters in the database as well

Workout Tracking

This screen allows users to log their workouts. It includes a form to enter workout details (name, reps, sets) and a list of recently logged activities. The interface is designed for quick and easy data entry, and adds all this information to the firebase database.

Workout Planning

A comprehensive view of available workout plans, including both preset and user-created options. Each plan is displayed with a title, brief description, and excercises involved

Calorie Tracking

A detailed page for monitoring daily calorie intake. It features a simple input method for adding foods consumed, a running total of calories for the day, and a visual representation of progress towards the daily goal.

Community Page

An engaging interface showcasing current community challenges. It displays challenge titles, descriptions, participant counts, and deadlines. Users can easily join challenges with a single tap, and the screen shows which challenges the user is currently participating in.
